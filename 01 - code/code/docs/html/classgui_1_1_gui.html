Description détaillée

Le Gui gère l'ensemble des interactions homme-machine du jeu. D'un coté un système de fenêtre, boutons basiques gérant les différents menus de l'appli. De l'autre, le menu phéromones, système central de l'interaction du joueur avec ses fourmis.

Gère les gadgets composant les interfaces. Ce n'est pas un singleton mais il ne doit y avoir qu'une seul instance.
